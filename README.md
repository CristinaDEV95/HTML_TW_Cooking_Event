# The Culinary Crucible

Aplicación web simple para gestionar ingredientes y descubrir qué recetas pueden prepararse según el stock disponible. Funciona completamente en el navegador y guarda los datos de manera local.

## Funcionalidades
- Selección de ingredientes con íconos.
- Cálculo automático de recetas disponibles.
- Descuento de ingredientes al cocinar.
- Guardado automático en `localStorage`.
- Interfaz responsiva y ligera.
- Compatible con GitHub Pages.

## Tecnologías
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage

## Estructura
- index.html
- style.css
- script.js
- script.js
- recipes.js

## Uso
1. Selecciona ingredientes y cantidades.
2. El sistema muestra recetas posibles.
3. Cocina una receta para descontar stock.
4. Los datos quedan guardados localmente.
